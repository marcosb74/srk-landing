# Sin Ryong Kwan - Academia de Taekwondo

Landing page para la academia de taekwondo Sin Ryong Kwan, construida con [Astro](https://astro.build).

## Getting Started

Primero, instala las dependencias:

```bash
npm install
```

Luego, ejecuta el servidor de desarrollo:

```bash
npm run dev
```

Abre [http://localhost:4321](http://localhost:4321) en tu navegador para ver el resultado.

Puedes empezar a editar la página modificando `src/pages/index.astro`. La página se actualiza automáticamente mientras editas el archivo.

## Build

Para crear un build de producción:

```bash
npm run build
```

Para previsualizar el build:

```bash
npm run preview
```

## Tecnologías

- [Astro](https://astro.build) - Framework web moderno
- [React](https://react.dev) - Para componentes interactivos
- [Tailwind CSS](https://tailwindcss.com) - Para estilos
- [Geist Font](https://vercel.com/font) - Fuente tipográfica

## Estructura del Proyecto

```
/
├── public/          # Archivos estáticos (imágenes, iconos, etc.)
├── src/
│   ├── components/  # Componentes React y Astro
│   ├── layouts/      # Layouts de Astro
│   ├── pages/        # Páginas (routing automático)
│   ├── styles/       # Estilos globales
│   └── lib/          # Utilidades
└── package.json
```
